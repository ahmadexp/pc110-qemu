# Disks

Boot media. These contain copyrighted IBM software (PC DOS / Personaware) and
are **not** included in this repository — supply your own.

| File | How to get it |
| --- | --- |
| `Personaware-disk.img` | Bootable disk built from a raw PC110 partition dump: `python3 tools/make-disk.py your-partition-dump.img disks/Personaware-disk.img` |
| `easysetup-floppy.img` | Generated by `boot/build-floppy.sh` from `roms/pc110_bios.bin` (Easy-Setup loader + unpacked image). |

If your Personaware dump is already a full disk image (has an MBR / boots on its
own), you can use it directly as `Personaware-disk.img` and skip `make-disk.py`.

## CONFIG.SYS tweak for QEMU

The stock PC110 `CONFIG.SYS` puts the EMS page frame at `FRAME=E000`, which
collides with SeaBIOS on QEMU and freezes the boot. After building the disk,
move the frame out of the way (via `mtools`, using the 32-sector partition
offset):

```
mcopy -i disks/Personaware-disk.img@@16384 ::CONFIG.SYS /tmp/config.sys
# edit the EMM386.EXE line, e.g.:
#   DEVICE=C:\DOS\EMM386.EXE RAM I=B000-B7FF X=DC00-DFFF X=E000-EFFF FRAME=CC00
mdel  -i disks/Personaware-disk.img@@16384 ::CONFIG.SYS
mcopy -i disks/Personaware-disk.img@@16384 /tmp/config.sys ::CONFIG.SYS
```
